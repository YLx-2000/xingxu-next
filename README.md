# 星叙 Next.js 原型项目

这是一个基于 Next.js + Tailwind CSS 的多页面静态原型项目，适合本地运行与继续开发。

## 本地运行

```bash
npm install
npm run dev
```

然后打开：

```bash
http://localhost:3000
```

## 页面说明

- `/` 首页：品牌介绍、平台优势、参与引导
- `/publish` 发布创意页：结构化填写创意内容
- `/square` 创意广场：浏览和筛选创意
- `/idea/3` 创意详情页：查看创意、回复、留言互动
- `/admin` 简单后台页：审核、推荐、内容管理
